# “说走就走”MVP 需求文档

## 目标与指标

用户在获得临时假期后，通过当季推荐、目的地详情和交通比较，在三步内形成可执行的国内出行选择。核心指标为推荐到详情点击率、交通查询完成率、攻略收藏率与价格刷新率。

## 用户流程

1. 用户按天数、预算与偏好筛选当季榜单。
2. 用户查看热门体验、建议路线和出发须知。
3. 用户输入出发地、日期与人数，比较三类交通方案。
4. 用户使用手机号登录并收藏目的地快照及交通参考方案。
5. 用户在出发前刷新价格，并跳转持牌平台核价或购票。

## 功能边界

MVP 仅覆盖中国大陆境内旅行，不提供站内购票、支付、评论、社交发布或复杂行程编辑。管理能力首版由数据库种子和受控内容维护流程承担，正式运营后台作为下一阶段模块。

## 内容与交通规则

榜单分数由季节适宜度、公开趋势、交通便利度和预算友好度形成。内容必须是原创摘要、公共事实或已授权素材。交通方案必须显示来源、更新时间和是否仅供参考；无授权实时接口时不得宣传为实时最低价。

## 验收标准

- 320px 至 560px 宽度下三个 Tab 可正常操作。
- 三种交通策略对价格与舒适度明显不同的样本产生不同首选。
- 未登录收藏会引导登录，登录后收藏可查询、删除和刷新。
- Provider 不可用时可使用明确标注的参考数据，不伪装实时结果。
- 类型检查、单元测试和生产构建通过。
